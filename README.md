# Nick Cinera Resume

Public resume artifacts for Nick Cinera, focused on trading systems, low-latency UI, full-stack capital-markets engineering, and AI-assisted engineering workflows.

## Files

- `docs/index.html` - static web resume for GitHub Pages.
- `output/Nick_Cinera_Resume.pdf` - polished one-page resume for applications and recruiter sharing.
- `output/Nick_Cinera_Resume.docx` - editable Word version.
- `output/Nick_Cinera_Resume.md` - source text version.

## Web Version

The static web resume is designed for GitHub Pages:

- Pages URL, once enabled: `https://extrieve.github.io/nick-resume/`
- GitHub-rendered Markdown fallback: `https://github.com/Extrieve/nick-resume/blob/main/output/Nick_Cinera_Resume.md`

To enable the Pages URL, set the repository's GitHub Pages source to `main` / `docs` in repository Settings > Pages.

## Positioning

The resume emphasizes production ownership of real-time trading platforms at Goldman Sachs, including React/TypeScript front ends, Java/WebSocket streaming services, AG Grid-heavy workflows, event-driven architecture, canary migrations, CI/CD release discipline, and pragmatic use of AI and agentic coding systems.
